## 摘要
探索仍然是利用强化学习训练的大语言模型智能体的关键瓶颈。虽然先前的方法利用了预训练知识，但在需要发现新状态的环境中表现不佳。我们提出了探索性记忆增强同策略与异策略优化（EMPO$^2$），这是一种混合强化学习框架，它利用记忆进行探索，并结合了同策略和异策略更新，使大语言模型在有记忆的情况下表现出色，同时确保在无记忆情况下的鲁棒性。在 ScienceWorld 和 WebShop 上，EMPO$^2$ 较 GRPO 分别实现了 128.6% 和 11.3% 的提升。此外，在分布外测试中，EMPO$^2$ 展示了对新任务卓越的适应能力，仅需少量带有记忆的尝试且无需参数更新。这些结果表明，EMPO$^2$ 是一个构建更具探索性和泛化能力的大语言模型智能体的极具前景的框架。

## Abstract
Exploration remains the key bottleneck for large language model agents trained with reinforcement learning. While prior methods exploit pretrained knowledge, they fail in environments requiring the discovery of novel states. We propose Exploratory Memory-Augmented On- and Off-Policy Optimization (EMPO$^2$), a hybrid RL framework that leverages memory for exploration and combines on- and off-policy updates to make LLMs perform well with memory while also ensuring robustness without it. On ScienceWorld and WebShop, EMPO$^2$ achieves 128.6% and 11.3% improvements over GRPO, respectively. Moreover, in out-of-distribution tests, EMPO$^2$ demonstrates superior adaptability to new tasks, requiring only a few trials with memory and no parameter updates. These results highlight EMPO$^2$ as a promising framework for building more exploratory and generalizable LLM-based agents.

# 论文总结：基于混合同策略与异策略优化的探索性记忆增强 LLM 智能体

## 1. 核心问题与整体含义（研究动机和背景）
论文聚焦于 **LLM 智能体在强化学习（RL）训练中的“探索瓶颈”**。
*   **研究动机**：现有的 LLM 智能体在 RL 训练中往往过度依赖预训练的先验知识（Exploitation），而在需要发现新状态或主动获取新信息的复杂环境中（如科学实验、多步规划），其探索能力（Exploration）严重不足。
*   **背景问题**：虽然引入外部记忆（如 Reflexion）可以帮助智能体纠错，但静态参数限制了其持续改进的能力；而传统的在线 RL（如 GRPO）在缺乏有效引导时容易陷入局部最优。
*   **核心目标**：建立一个统一框架，通过联合更新 LLM 的**参数化策略**和**非参数化记忆**，使智能体既能利用记忆进行高效探索，又能将这些经验“内化”到模型参数中。

## 2. 方法论：核心思想与关键技术
EMPO² 提出了一种混合优化框架，其核心在于**“记忆引导探索”与“知识内化蒸馏”**：

*   **双模式 Rollout（采样阶段）**：
    *   **无记忆提示**：仅根据当前状态和任务生成动作。
    *   **记忆增强提示**：从记忆库中检索相关的“提示（Tips）”（由模型自身对过去失败/成功的反思生成），引导模型尝试新路径。
*   **双模式 Update（更新阶段）**：
    *   **同策略（On-Policy）更新**：直接优化带记忆或不带记忆的轨迹。
    *   **异策略（Off-Policy）更新（核心创新）**：将“带记忆引导”下产生的成功轨迹作为教师，通过重要性采样（Importance Sampling）更新“不带记忆”的基座模型。这本质上是一种**奖励引导的知识蒸馏**，使模型在推理时不依赖外部记忆也能具备探索后的高性能。
*   **关键技术细节**：
    *   **自生成记忆**：模型在每轮结束后自我反思生成自然语言 Tips。
    *   **稳定化技术**：引入 **Token 掩码机制**，过滤掉低概率 Token 以防止异策略更新时的梯度爆炸。
    *   **内在奖励（Intrinsic Rewards）**：基于状态新颖性（余弦相似度）给予奖励，鼓励模型访问未见过的状态。

*   **Benchmark 与对比方法**：
    *   **基准模型**：Qwen2.5-7B-Instruct。
    *   **对比方法**：非参数化 RL（Reflexion）、离线 RL（Retrospex）、在线 RL（GRPO、GiGPO）。
*   **测试维度**：包括分布内（ID）任务性能、分布外（OOD）任务的零样本/少样本迁移能力。

*   **训练细节**：基于 `verl` 和 `agent-lightning` 框架开发。ScienceWorld 任务限制每步 32 token，总长度 4500 token，最大 30 步。

## 5. 实验数量与充分性
*   **实验规模**：在 ScienceWorld 的 19 个不同类别的任务上进行了详尽测试，并在 WebShop 上验证了通用性。
*   **消融实验**：针对同/异策略组合、内在奖励机制、记忆检索概率 $p$ 和异策略更新概率 $q$ 进行了多组消融实验。
*   **客观性**：实验对比了当前最先进的在线 RL 算法（如 GRPO），并展示了训练曲线。通过 OOD 实验验证了模型并非仅仅死记硬背，而是学会了“如何利用记忆进行探索”。整体实验设计严谨，覆盖面广。

## 6. 主要结论与发现
*   **性能飞跃**：在 ScienceWorld 上比 GRPO 提升了 **128.6%**，在 WebShop 上提升了 **11.3%**。
*   **解决停滞问题**：GRPO 在复杂任务（如电力组件）中常因探索不足导致性能过早平跨，而 EMPO² 能持续改进并最终解决任务。
*   **强大的 OOD 适应性**：经过 EMPO² 训练的模型在面对全新任务时，仅需通过几轮带有记忆的尝试（无需更新参数），就能快速适应并超越基准模型。
*   **内化有效性**：证明了通过异策略蒸馏，可以将外部记忆中的逻辑成功转化为模型自身的参数化能力。

## 7. 优点（亮点）
*   **闭环自进化**：模型自己生成 Tips、自己探索、自己蒸馏，减少了对人类专家数据或 GPT-4 引导的依赖。
*   **混合优化机制**：巧妙结合了同策略的稳定性和异策略的探索效率，解决了 LLM 在 RL 中容易“迷失”的问题。
*   **鲁棒性**：最终模型在推理时不需要外部记忆也能保持高水平，降低了部署成本。

## 8. 不足与局限
*   **计算开销**：引入记忆机制增加了 Rollout 阶段的耗时（约增加 19% 的时间），主要用于 Tips 的生成和存储。
*   **检索机制尚浅**：目前仅使用简单的余弦相似度进行记忆检索，在海量记忆处理上可能存在瓶颈。
*   **模型规模局限**：实验主要集中在 7B 参数模型，尚未在大规模模型（如 70B+）或多模态模型上进行广泛验证。
*   **任务领域**：虽然在科学和购物场景有效，但在代码、数学等逻辑严密但环境反馈稀疏的领域仍需进一步探索。
