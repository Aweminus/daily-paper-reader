## 摘要
具身视觉追踪（EVT）是具身智能中的核心动态任务，要求智能体精确跟随语言指定的任务目标。然而，现有的大多数方法依赖于单智能体模仿学习，面临专家数据成本高昂以及由于静态训练环境导致的泛化能力有限等问题。受竞争驱动的能力演化启发，我们提出了 CoMaTrack，这是一个竞争性博弈论多智能体强化学习框架，在具有竞争性子任务的动态对抗环境中训练智能体，从而产生更强的自适应规划和抗干扰策略。我们进一步推出了 CoMaTrack-Bench，这是首个基于 Habitat 的开源基准协议和剧集集，用于具有动态对决特征的语言条件竞争性 EVT。它包含追踪者与自适应对手在多样化环境和指令下的博弈场景，能够实现在主动对抗交互下的标准化鲁棒性评估。实验表明，CoMaTrack 在标准基准测试和 CoMaTrack-Bench 上均取得了当前最先进的结果。值得注意的是，使用我们框架训练的 3B VLM 在具有挑战性的 EVT-Bench 上超越了此前基于 7B 模型的单智能体模仿学习方法，在 STT、DT 和 AT 指标上分别达到了 92.1%、74.2% 和 57.5%。基准测试代码将在 https://github.com/wlqcode/CoMaTrack-Bench 提供。

## Abstract
Embodied Visual Tracking (EVT), a core dynamic task in embodied intelligence, requires an agent to precisely follow a language-specified target. Yet most existing methods rely on single-agent imitation learning, suffering from costly expert data and limited generalization due to static training environments. Inspired by competition-driven capability evolution, we propose CoMaTrack, a competitive game-theoretic multi-agent reinforcement learning framework that trains agents in a dynamic adversarial setting with competitive subtasks, yielding stronger adaptive planning and interference-resilient strategies. We further introduce CoMaTrack-Bench, the first open-source Habitat-based benchmark protocol and episode set for language-conditioned competitive EVT featuring dynamic dueling, featuring game scenarios between a tracker and adaptive opponents across diverse environments and instructions, enabling standardized robustness evaluation under active adversarial interactions. Experiments show that CoMaTrack achieves state-of-the-art results on both standard benchmarks and CoMaTrack-Bench. Notably, a 3B VLM trained with our framework surpasses previous single-agent imitation learning methods based on 7B models on the challenging EVT-Bench, achieving 92.1% in STT, 74.2% in DT, and 57.5% in AT. The benchmark code will be available at https://github.com/wlqcode/CoMaTrack-Bench.

    *   **模仿学习（IL）的局限**：现有方法多依赖专家演示数据，不仅采集成本高，且在面对分布外（OOD）场景或动态干扰时表现脆弱。
    *   **单智能体强化学习（RL）的不足**：在静态环境或固定目标下训练，智能体缺乏足够的挑战来演化出复杂的规划和抗干扰策略。
*   **核心动机**：受生物竞争进化启发，通过引入**竞争性多智能体博弈**，让智能体在对抗中自动形成“难度递增”的课程学习，从而提升策略的鲁棒性和泛化性。

### 2. 方法论
*   **核心思想**：将 EVT 建模为追踪者（Tracker）与对手（Opponent）之间的竞争博弈。对手通过遮挡、路径阻挡或争夺目标来增加任务难度，迫使追踪者学习预测和重新定位能力。
*   **关键技术细节**：
    *   **VLA 架构**：基于 **Qwen2.5VL-3B** 多模态大模型，结合 **Flow-matching** 动作头，直接输出连续的 5 步运动航点（Waypoints）。
    *   **两阶段训练**：
        1.  **监督微调（SFT）**：在导航、追踪和 VQA 混合数据集上进行预训练，建立基础的感知与指令遵循能力。
        2.  **多智能体强化学习（MARL）**：使用 **GRPO（组相对策略优化）** 算法。追踪者和对手均从 SFT 模型初始化，但在训练中采用非对称奖励函数。
    *   **非对称奖励设计**：
        *   **对手奖励**：鼓励其以更近的距离（1.25m）接近目标，制造干扰。
        *   **追踪者奖励**：鼓励其保持安全跟随距离（2.25m），同时惩罚与对手的碰撞。

### 3. 实验设计
*   **数据集与场景**：
    *   **EVT-Bench**：包含单目标追踪（STT）、干扰追踪（DT）和歧义追踪（AT）三个子任务。

### 6. 主要结论与发现
*   **竞争驱动进化**：多智能体对抗训练能显著降低碰撞率（从 3.1% 降至 0.9%）并提高追踪成功率。

### 7. 优点与亮点
*   **范式创新**：首次将竞争性博弈论 RL 引入基于 VLA 的具身追踪任务，打破了过度依赖专家数据的现状。

*   **基准贡献**：开源了 CoMaTrack-Bench，填补了具身追踪领域缺乏主动对抗评估协议的空白。

*   **任务局限性**：目前主要集中在追踪任务，尚未在更广泛的具身任务（如复杂物体搬运、长程多级指令导航）上进行大规模验证。

*   **算力门槛**：虽然 RL 阶段算力需求较低，但 SFT 阶段仍需要大规模 GPU 集群支持。
