## 摘要
检索增强生成 (RAG) 通过外部视觉知识扩展了大视觉语言模型 (LVLM)。然而，现有的视觉 RAG 系统通常依赖通用的检索信号，忽略了复杂推理所必需的细粒度视觉语义。为了解决这一局限性，我们提出了 UniDoc-RL，这是一个统一的强化学习框架，其中 LVLM 智能体共同执行检索、重排序、主动视觉感知和推理。UniDoc-RL 将视觉信息获取建模为具有层级化动作空间的序列决策问题。具体而言，它将视觉证据从粗粒度的文档检索逐步细化为细粒度的图像选择和主动区域裁剪，使模型能够抑制无关内容并关注信息密集区域。为了实现有效的端到端训练，我们引入了一种稠密多奖励方案，为每个动作提供任务感知的监督。基于群体相对策略优化 (GRPO)，UniDoc-RL 在不依赖独立价值网络的情况下，使智能体行为与多个目标保持一致。为了支持这种训练范式，我们构建了一个包含细粒度动作标注的高质量推理轨迹综合数据集。在三个基准测试上的实验表明，UniDoc-RL 持续超越了最先进的基准模型，相比之前的基于强化学习的方法提升高达 17.7%。

## Abstract
Retrieval-Augmented Generation (RAG) extends Large Vision-Language Models (LVLMs) with external visual knowledge. However, existing visual RAG systems typically rely on generic retrieval signals that overlook the fine-grained visual semantics essential for complex reasoning. To address this limitation, we propose UniDoc-RL, a unified reinforcement learning framework in which an LVLM agent jointly performs retrieval, reranking, active visual perception, and reasoning. UniDoc-RL formulates visual information acquisition as a sequential decision-making problem with a hierarchical action space. Specifically, it progressively refines visual evidence from coarse-grained document retrieval to fine-grained image selection and active region cropping, allowing the model to suppress irrelevant content and attend to information-dense regions. For effective end-to-end training, we introduce a dense multi-reward scheme that provides task-aware supervision for each action. Based on Group Relative Policy Optimization (GRPO), UniDoc-RL aligns agent behavior with multiple objectives without relying on a separate value network. To support this training paradigm, we curate a comprehensive dataset of high-quality reasoning trajectories with fine-grained action annotations. Experiments on three benchmarks demonstrate that UniDoc-RL consistently surpasses state-of-the-art baselines, yielding up to 17.7% gains over prior RL-based methods.

*   **核心问题**：现有的视觉 RAG 系统主要依赖通用的检索信号（如向量相似度），难以捕捉复杂推理所需的细粒度视觉语义。此外，视觉文档（如图表、扫描件）包含大量冗余背景，直接输入全图会导致模型受噪声干扰。
*   **研究动机**：
    1.  **检索不精准**：粗粒度检索常引入无关上下文，导致模型产生幻觉。
    2.  **视觉利用率低**：模型通常被动接收全图，缺乏像人类一样主动关注局部关键区域的能力。
    3.  **优化困难**：现有的 RL 方法多采用稀疏的“结果奖励”，难以对检索、裁剪等中间决策步骤进行有效的信用分配（Credit Assignment）。

### 2. 方法论
UniDoc-RL 将视觉 RAG 建模为一个**由粗到精（Coarse-to-Fine）**的序列决策过程，核心技术包括：
*   **层级化动作空间（Hierarchical Action Space）**：
    1.  **图像搜索（Search）**：调用外部工具进行初步的大规模候选文档检索。
    2.  **精确选择（Select）**：由 LVLM 智能体对候选图进行语义重排序，过滤无关噪声。
    3.  **视觉感知（Perceive）**：通过 `<bbox>` 动作主动定位关键区域，执行裁剪和缩放（Crop & Zoom），获取高分辨率局部细节。
*   **密集多奖励机制（Dense Multi-Rewards）**：
    *   **检索奖励 ($r_{ir}$)**：基于 NDCG 指标评估搜索结果的质量。
    *   **选择奖励 ($r_{sel}$)**：评估模型选中的图像是否包含真实答案（Golden Image）。
    *   **感知奖励 ($r_{crop}$)**：基于预测框与真实区域的交并比（IoU）给予反馈。
    *   **模式奖励 ($r_{pat}$)**：确保模型遵循预定义的 XML 格式。
    *   **结果奖励 ($r_{ans}$)**：由奖励模型（RM）评估最终答案的正确性。
*   **算法流程**：采用 **GRPO（群体相对策略优化）** 算法进行端到端训练。该算法通过组内样本的相对得分来估计优势函数，无需独立的价值网络（Value Network），节省了显存。

### 6. 主要结论与发现
*   **性能大幅提升**：UniDoc-RL 在所有基准上均优于现有方法，相比之前的 RL 方法（VRAG-RL），在 3B 和 7B 模型上分别实现了 **17.5% 和 17.7%** 的性能增益。
*   **RL 改变行为模式**：相比 SFT 模型，经过 RL 训练的模型表现出更积极的主动信息获取行为（裁剪频率显著增加），且裁剪精度更高。
*   **解决语义鸿沟**：LVLM 驱动的“精确选择”动作显著提高了检索的召回率，弥补了传统检索器与下游推理任务之间的语义断层。

### 7. 优点（亮点）
*   **由粗到精的逻辑**：将 RAG 流程拆解为层级化动作，符合人类处理复杂视觉文档的直觉。
*   **密集奖励设计**：通过为中间步骤（搜索、选择、裁剪）提供显式监督，有效解决了长链条推理中的信用分配难题。
*   **高质量数据集**：构建并开源了一个包含 1.2 万条 SFT 轨迹和 5.5 千条 RL 轨迹的精细标注数据集，对社区有重要贡献。

### 8. 不足与局限
*   **数据合成依赖**：训练轨迹高度依赖于更强大的教师模型（如 Qwen3-VL-235B），这可能导致学生模型性能受限于教师模型的上限。
*   **推理延迟**：由于引入了多轮“搜索-选择-感知”的迭代过程，系统的端到端推理延迟（Latency）会高于传统的单次 RAG。
*   **应用限制**：目前主要针对静态文档图像，对于动态视频或极大规模实时网页 RAG 的适用性尚待验证。
