## 摘要
智能体技能是程序性知识和可执行资源的结构化包，智能体在推理时动态加载这些资源，已成为增强大语言模型（LLM）智能体的一种可靠机制。然而，推理时的技能增强存在根本性的局限：检索噪声会引入无关的引导，注入的技能内容会带来巨大的 Token 开销，且模型从未真正习得它仅仅是在遵循的知识。我们探究技能是否可以被内化到模型参数中，从而在无需任何运行时技能检索的情况下实现零样本自主行为。我们提出了 SKILL0，这是一个专为技能内化设计的上下文强化学习框架。SKILL0 引入了一种训练阶段的课程学习方案，从完整的技能上下文开始并逐步撤出。技能在离线状态下按类别分组，并与交互历史一起渲染成紧凑的可视化上下文，教导模型进行工具调用和多轮任务完成。随后，动态课程评估每个技能文件的同策略帮助程度，在随线性衰减的预算内，仅保留当前策略仍能从中获益的技能，直到智能体在完全零样本的设置下运行。广泛的智能体实验表明，SKILL0 相比标准强化学习基线取得了显著提升（ALFWorld 提升 9.7%，Search-QA 提升 6.6%），同时保持了每步少于 0.5k Token 的高效上下文。我们的代码可在 https://github.com/ZJU-REAL/SkillZero 获取。

## Abstract
Agent skills, structured packages of procedural knowledge and executable resources that agents dynamically load at inference time, have become a reliable mechanism for augmenting LLM agents. Yet inference-time skill augmentation is fundamentally limited: retrieval noise introduces irrelevant guidance, injected skill content imposes substantial token overhead, and the model never truly acquires the knowledge it merely follows. We ask whether skills can instead be internalized into model parameters, enabling zero-shot autonomous behavior without any runtime skill retrieval. We introduce SKILL0, an in-context reinforcement learning framework designed for skill internalization. SKILL0 introduces a training-time curriculum that begins with full skill context and progressively withdraws it. Skills are grouped offline by category and rendered with interaction history into a compact visual context, teaching he model tool invocation and multi-turn task completion. A Dynamic Curriculum then evaluates each skill file's on-policy helpfulness, retaining only those from which the current policy still benefits within a linearly decaying budget, until the agent operates in a fully zero-shot setting. Extensive agentic experiments demonstrate that SKILL0 achieves substantial improvements over the standard RL baseline (+9.7\% for ALFWorld and +6.6\% for Search-QA), while maintaining a highly efficient context of fewer than 0.5k tokens per step. Our code is available at https://github.com/ZJU-REAL/SkillZero.

### 1. 核心问题与研究动机
*   **核心问题**：当前的 LLM 智能体主要依赖“推理时技能增强”（Inference-time Skill Augmentation），即通过检索技能库并将其注入 Prompt 来引导智能体。这种方式存在三大缺陷：
    1.  **检索噪声**：引入无关引导干扰上下文。
    2.  **Token 开销**：多轮对话中注入技能描述会导致 Token 消耗巨大，限制扩展性。
    3.  **知识未内化**：模型只是在“遵循”指令而非“掌握”技能，能力存在于上下文中而非参数中。
*   **研究动机**：探索是否能通过强化学习（RL）将外部技能“内化”到模型参数中，实现“训练时有技能，推理时零技能（Zero-shot）”的自主行为。

### 2. 方法论
SKILL0 提出了**上下文智能体强化学习（In-Context Agentic RL）**框架，核心思想是在训练中提供技能引导，通过动态课程学习逐步撤除引导，迫使模型将策略固化至参数。

*   **核心技术细节**：
    *   **上下文渲染与压缩（Context Rendering）**：为了降低 Token 成本，将文本历史和技能描述渲染为紧凑的 RGB 图像，通过视觉编码器转化为视觉 Token。模型可自适应生成压缩率 $c_t$。
    *   **上下文强化学习（ICRL）**：在训练 Rollout 阶段提供技能上下文，但在推理阶段完全移除。优化目标结合了任务成功率和上下文压缩效率。
    *   **动态课程学习（Dynamic Curriculum）**：
        1.  **相关性驱动分组**：离线将技能按类别分组，并关联特定的验证子任务。
        2.  **帮助度驱动退火**：定义帮助度指标 $\Delta_k = Acc_{有技能} - Acc_{无技能}$。
        3.  **线性预算衰减**：设定技能预算 $M(s)$ 随训练阶段线性减少。
        4.  **过滤与排序**：每隔 $d$ 步评估技能，仅保留当前策略仍需依赖（$\Delta_k > 0$）且表现最好的 Top-M 个技能，直至预算归零。

### 5. 实验数量与充分性
*   **实验规模**：涵盖了从 3B 到 7B 的模型规模，在两个完全不同的领域（具身控制与知识搜索）进行了验证。
*   **消融实验**：针对技能预算策略（如固定全量 vs 线性衰减）、动态课程组件（过滤、排序、选择）、验证间隔 $d$ 等进行了详细消融。

### 6. 主要结论与发现
*   **性能卓越**：SKILL0 在 ALFWorld 上比 AgentOCR 提升了 **9.7%**，在 Search-QA 上提升了 **6.6%**。
*   **高效推理**：由于推理时无需检索技能且使用了视觉压缩，每步 Token 开销**低于 0.5k**，比 SkillRL 等方法降低了 5 倍以上。
*   **内化效应**：实验观察到“帮助度”曲线呈现先升后降的趋势，证实了模型在训练后期即使没有外部技能引导也能保持高成功率，实现了真正的知识内化。
*   **泛化能力**：在未见过的多跳问答数据集（如 Bamboogle）上表现出极强的零样本泛化能力。

### 7. 优点（亮点）
*   **范式创新**：首次将技能内化定义为明确的强化学习训练目标，解决了智能体对外部 Prompt 的过度依赖。
*   **动态自适应**：提出的动态课程学习不是死板的时间表，而是根据模型当前的“学习状态”（帮助度）来决定何时撤走“脚手架”。
*   **极高效率**：结合视觉编码压缩和零技能推理，极大地优化了长程任务的推理成本。

### 8. 不足与局限
*   **依赖初始技能库**：SKILL0 的效果高度依赖于初始 SkillBank 的质量，如果初始技能有误，可能会导致错误的内化。
*   **离线分组开销**：相关性驱动的技能分组需要一定的先验知识或预处理工作，迁移到全新领域时可能存在人工成本。
*   **视觉编码器限制**：虽然视觉压缩节省了 Token，但对于极度精细的文本信息，视觉编码可能存在信息丢失的风险（尽管实验中表现良好）。
