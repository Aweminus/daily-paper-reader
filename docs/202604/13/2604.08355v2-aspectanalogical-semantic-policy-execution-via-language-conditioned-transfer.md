## 摘要
强化学习（RL）智能体通常难以将知识泛化到新任务中，即使这些任务与它们已经掌握的任务在结构上相似。尽管最近的方法尝试通过零样本迁移（zero-shot transfer）来缓解这一问题，但它们通常受限于预定义的离散类别系统，从而限制了其对新颖或组合式任务变体的适应能力。我们提出了一种更具普适性的方法，通过文本条件的变分自编码器（VAE），将离散隐变量替换为自然语言条件。我们的核心创新在于测试时利用大语言模型（LLM）作为动态的“语义算子”（semantic operator）。我们的智能体不再依赖僵化的规则，而是通过查询 LLM 对当前观测的描述进行语义重映射，使其与源任务对齐。这种与源任务对齐的描述作为条件引导 VAE 生成一个与智能体原始训练兼容的想象状态，从而实现策略的直接复用。通过利用 LLM 灵活的推理能力，我们的方法在广泛的复杂且真正新颖的类比任务中实现了零样本迁移，超越了固定类别映射的局限性。代码和视频可在 https://anonymous.4open.science/r/ASPECT-85C3/ 获取。

## Abstract
Reinforcement Learning (RL) agents often struggle to generalize knowledge to new tasks, even those structurally similar to ones they have mastered. Although recent approaches have attempted to mitigate this issue via zero-shot transfer, they are often constrained by predefined, discrete class systems, limiting their adaptability to novel or compositional task variations. We propose a significantly more generalized approach, replacing discrete latent variables with natural language conditioning via a text-conditioned Variational Autoencoder (VAE). Our core innovation utilizes a Large Language Model (LLM) as a dynamic \textit{semantic operator} at test time. Rather than relying on rigid rules, our agent queries the LLM to semantically remap the description of the current observation to align with the source task. This source-aligned caption conditions the VAE to generate an imagined state compatible with the agent's original training, enabling direct policy reuse. By harnessing the flexible reasoning capabilities of LLMs, our approach achieves zero-shot transfer across a broad spectrum of complex and truly novel analogous tasks, moving beyond the limitations of fixed category mappings. ### 2. 方法论
ASPECT 框架的核心思想是将 LLM 作为“语义算子”，结合文本条件变分自编码器（VAE）进行状态重构。
*   **文本条件 VAE**：
    *   **编码器**：将当前观察 $s_t$ 编码为空间隐变量 $z$，捕捉环境的结构化特征（如物体位置、布局）。
    *   **解耦训练**：通过对抗训练（梯度反转层 GRL）确保 $z$ 与语义文本信息正交，即 $z$ 只包含结构信息，不包含语义身份。
    *   **解码器**：利用 FiLM（特征线性调制）和交叉注意力机制，结合文本嵌入来重构图像。
*   **LLM 语义算子**：
    *   在测试时，首先获取当前状态的描述（通过 VLM 或模板）。
    *   LLM 根据源任务和目标任务的上下文，将描述中的新物体/环境重映射为源任务中的对应物（例如：将“黄色小鸭”重映射为“蓝色方块”）。
*   **类比执行流程**：
    1.  观察目标状态 $s_t$ 并生成文本描述。
    2.  LLM 将描述转换为“源任务对齐”的描述。
    3.  VAE 解码器结合原始结构特征 $z$ 和重映射后的描述，生成“想象中的源状态” $s_{imagined}$。
    4.  预训练的源策略 $\pi_{source}$ 直接作用于 $s_{imagined}$ 并输出动作。

*   **语义灵活性**：通过 LLM 的推理，智能体能够处理复杂的语义对齐（如将物理属性“脆性”进行类比），这是传统离散类别方法无法做到的。

### 7. 优点
*   **算法无关性**：可适配 DQN、PPO、SAC 等多种 RL 算法，具有极强的通用性。
*   **开放世界潜力**：摆脱了固定类别标签的束缚，能够利用 LLM 的先验知识处理训练时未见的物体。
*   **解耦架构**：通过将结构特征与语义特征分离，保证了“想象”过程中的空间一致性。

### 8. 不足与局限
*   **生成伪影**：VAE 在处理极端近景或复杂背景时可能产生伪影，导致策略失效。
*   **LLM 风险**：系统性能高度依赖 LLM 的推理质量，LLM 的幻觉、偏见或错误映射可能导致不可预测的智能体行为。
*   **实时性挑战**：在每个时间步调用 LLM 和 VAE 进行推理会带来显著的计算延迟，限制了其在高速实时控制场景中的应用。
*   **安全验证**：缺乏对 LLM 生成的语义映射进行自动化安全验证的机制。
