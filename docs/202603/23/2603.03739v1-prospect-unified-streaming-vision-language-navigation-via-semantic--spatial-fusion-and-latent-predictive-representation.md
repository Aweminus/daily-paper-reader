## 摘要
多模态大语言模型（MLLMs）推动了零样本端到端视觉语言导航（VLN）的发展，但鲁棒的导航不仅需要语义理解，还需要对环境动态和空间结构进行预测建模。我们提出了 PROSPECT，这是一种统一的流式导航智能体，它将流式视觉-语言-动作（VLA）策略与潜空间预测表示学习相结合。PROSPECT 使用 CUT3R 作为流式 3D 基础空间编码器，以生成长上下文、绝对尺度的空间特征，并通过交叉注意力机制将其与 SigLIP 语义特征进行融合。在训练过程中，我们引入了可学习的流查询令牌（stream query tokens），用于查询流式上下文并预测下一步的 2D 和 3D 潜空间特征（而非像素或显式模态），并在冻结的 SigLIP 和 CUT3R 教师模型的潜空间中进行监督。预测分支在不增加推理开销的情况下塑造了内部表示。在 VLN-CE 基准测试和真实机器人部署上的实验表明，该方法达到了最先进的性能，并在多种光照条件下提升了长时程鲁棒性。我们将很快向社区发布代码。

## Abstract
Multimodal large language models (MLLMs) have advanced zero-shot end-to-end Vision-Language Navigation (VLN), yet robust navigation requires not only semantic understanding but also predictive modeling of environment dynamics and spatial structure. We propose PROSPECT, a unified streaming navigation agent that couples a streaming Vision-Language-Action (VLA) policy with latent predictive representation learning. PROSPECT uses CUT3R as a streaming 3D foundation spatial encoder to produce long-context, absolute-scale spatial features, and fuses them with SigLIP semantic features via cross-attention. During training, we introduce learnable stream query tokens that query the streaming context and predict next-step 2D and 3D latent features (rather than pixels or explicit modalities), supervised in the latent spaces of frozen SigLIP and CUT3R teachers. The predictive branch shapes internal representations without inference overhead. Experiments on VLN-CE benchmarks and real-robot deployment demonstrate state-of-the-art performance and improved long-horizon robustness under diverse lighting. We will release code for the community soon.

### 2. 方法论
*   **核心思想**：在流式 VLA 架构中引入“流查询令牌（Stream Query Tokens）”，在训练阶段预测未来的 2D 语义和 3D 空间潜空间特征，从而在不增加推理成本的前提下强化模型内部的表征能力。
*   **关键技术细节**：
    *   **双流感知与融合**：使用 **SigLIP** 提取 2D 语义特征，使用 **CUT3R**（流式 3D 基础模型）提取具有绝对尺度的 3D 空间特征。两者通过交叉注意力机制（Cross-Attention）进行融合。
    *   **潜空间预测（Latent Prediction）**：借鉴 JEPA 思想，不预测原始像素，而是预测冻结的教师模型（SigLIP 和 CUT3R）输出的潜空间特征。
    *   **流查询令牌**：在 LLM 输入中附加可学习的 $q_{2D}$ 和 $q_{3D}$ 令牌，通过轻量化解码器重建下一帧特征。
    *   **流式注意力掩码（Streaming Attention Mask）**：设计了一种特殊的掩码，确保：1) 时间因果性；2) 不同回合的查询令牌相互隔离；3) 2D 与 3D 预测任务互不干扰，防止模态间的信息泄露。
*   **损失函数**：总损失由导航动作的交叉熵损失、2D 特征的余弦相似度损失和 3D 特征的均方误差（MSE）损失加权组成。

### 6. 主要结论与发现
*   **性能领先**：PROSPECT 在 R2R 和 RxR 基准上均达到了第一梯队水平，尤其在 RxR（长指令、长路径）上的提升更为显著。
*   **预测表征的价值**：引入潜空间预测任务能显著提升导航性能，且 2D 语义预测与 3D 空间预测具有互补性。
*   **空间编码器的重要性**：相比于 VGGT，具有绝对尺度的 CUT3R 在流式导航中更稳定，且推理延迟更低。
*   **鲁棒性**：模型在真实世界中表现出极强的环境适应性，尤其是在黄昏和夜晚等低光照室外场景下，依然能保持较高的成功率。

### 7. 优点
*   **高效性**：预测分支仅在训练时存在，推理时被移除，实现了“预测引导学习”而无额外推理开销。
*   **空间智能**：通过集成 3D 基础模型，解决了纯 2D 导航模型缺乏空间尺度感的问题。
*   **抗噪性**：潜空间监督避免了生成式世界模型中常见的像素级噪声干扰，使模型更关注物理动态。
*   **流式架构**：原生支持长上下文流式输入，适合处理复杂的连续环境。

### 8. 不足与局限
*   **算力需求高**：训练过程需要大量的 A800 算力，对于小型研究团队门槛较高。
*   **远程依赖**：真实机器人部署仍依赖远程服务器进行推理，对网络带宽和稳定性有一定要求，尚未完全实现端侧独立运行。
*   **数据依赖**：性能提升在很大程度上受益于大规模的增强数据集（如 ScaleVLN），在极小样本场景下的表现尚待进一步验证。
