## 摘要
在视觉指令数据上进行有监督微调（SFT）通常会提升视觉-语言模型（VLM）的感知能力，但同时会降低其推理性能，从而在训练后阶段产生持续的“推理税”。我们研究了这种性能退化是否与深度表征访问受阻有关，并发现即使是固定的跨深度聚合也能显著恢复推理能力，这表明保留跨深度访问是 VLM 微调中一个重要的缺失因素。基于这一观察，我们提出了输入自适应深度聚合（IADA），这是一种轻量级机制，使跨深度检索具备输入自适应性和模态感知能力，并通过低秩瓶颈实现高效参数化。在 Qwen3-VL-2B 上，相比于仅使用 LoRA 的微调，IADA 仅增加了 0.14M 额外参数，便将平均推理分数提升了 9.5 分，平均感知分数提升了 3.3 分，且在参数高效的低秩设置下提升最为显著。

## Abstract
Supervised fine-tuning (SFT) on visual instruction data often improves perceptual capabilities in vision-language models (VLMs) while degrading reasoning performance, creating a persistent reasoning tax during post-training. We investigate whether this degradation is related to disrupted access to depth-wise representations, and find that even fixed cross-depth aggregation substantially restores reasoning, suggesting that preserved cross-depth access is an important missing factor in VLM fine-tuning. Building on this observation, we propose Input-Adaptive Depth Aggregation (IADA), a lightweight mechanism that makes cross-depth retrieval input-adaptive, modality-aware, and efficiently parameterized through a low-rank bottleneck. On Qwen3-VL-2B, IADA improves the average reasoning score by 9.5 points and the average perception score by $3.3$ points over LoRA-only fine-tuning with only 0.14M additional parameters, with the strongest gains appearing in parameter-efficient low-rank settings.

#### 1. 核心问题与整体含义（研究动机和背景）
视觉-语言模型（VLM）在经过视觉指令微调（SFT）后，通常会出现一种“推理税”（Reasoning Tax）现象：即模型的感知能力（如 OCR、物体识别）有所提升，但逻辑推理、数学和科学问题的处理能力却显著下降。
论文假设这种退化源于微调破坏了模型对预训练阶段形成的“深度特征层级”的访问——通常浅层编码感知模式，深层编码抽象推理。作者通过实验发现，仅需恢复对早期层表征的跨深度访问，即可大幅挽回推理损失。

#### 2. 方法论：输入自适应深度聚合（IADA）
IADA 是一种轻量级的模块，旨在微调期间显式建模跨层的信息路由。
*   **核心思想**：在 Transformer 块的边界插入聚合模块，根据当前输入动态地从之前的层中检索相关特征。
*   **关键技术细节**：
    *   **输入自适应 Query 生成**：对当前隐藏状态进行全局平均池化（MeanPool），通过一个低秩瓶颈结构（$W_{down}$ 和 $W_{up}$）生成针对当前输入的 Query 向量。
    *   **模态感知路由**：利用模态掩码将视觉和文本 Token 分开，分别为两种模态生成独立的 Query，并在各自的深度路径上进行交叉注意力（Cross-Attention）检索。
    *   **低秩瓶颈**：为了保持轻量化，Query 生成和检索过程均采用低秩参数化（如 Rank=16），仅需极少参数即可捕捉有效的路由逻辑。
    *   **门控残差注入**：检索到的跨深度特征通过一个初始值接近零的门控系数（$\sigma(\gamma)$）加回到当前隐藏状态，确保训练初期的稳定性。

*   **基座模型**：Qwen3-VL-2B-Instruct。
*   **微调数据**：LLaVA-Instruct-Mix（约 25.9 万条样本）。
*   **Benchmark（11个评测集）**：
    *   **推理类**：MMMU, MathVista, ScienceQA, MMStar, AI2D。
    *   **感知类**：MME, POPE, RealWorldQA, ChartQA, TextVQA, OCRBench。
*   **对比方法**：预训练基准（无微调）、标准 LoRA 微调、固定 Query 的注意力残差（AttnRes）+ LoRA。

#### 4. 资源与算力
*   **硬件**：使用单台 **NVIDIA H200 GPU**。
*   **训练配置**：采用 bf16 混合精度，开启梯度检查点（Gradient Checkpointing）。
*   **时长与规模**：训练 1 个 epoch，共 16,198 个优化步数，有效 Batch Size 为 16。LoRA 学习率为 $2 \times 10^{-5}$，IADA 学习率为 $1 \times 10^{-3}$。

#### 5. 实验数量与充分性
*   **实验规模**：作者在 11 个主流 Benchmark 上进行了全面测试，涵盖了从基础感知到高阶推理的多个维度。
*   **消融实验**：非常充分。作者对比了固定 Query vs. 自适应 Query、有无模态拆分、不同低秩 Rank（4 到 Full-rank）、Token 级 vs. 序列级路由、以及不同的门控机制。
*   **客观性**：实验设计了严谨的对照组（如固定 Query 诊断实验），证明了性能提升并非单纯来自参数量增加，而是来自“输入自适应”的结构设计。

#### 6. 主要结论与发现
*   **缓解推理税**：在 Qwen3-VL-2B 上，IADA + LoRA 相比纯 LoRA 微调，推理平均分提升了 **9.5 分**，感知平均分提升了 **3.3 分**。
*   **低秩高效性**：仅需 **0.14M** 的额外参数（Rank=16），效果甚至优于参数量大 60 倍的全秩（Full-rank）版本。
*   **自适应是关键**：输入自适应 Query 是性能提升的核心驱动力，而模态感知路由提供了进一步的精细化增益。
*   **与 PEFT 互补**：IADA 在低 Rank LoRA 设置下增益最明显，说明它提供了 LoRA 这种层内适配器无法实现的跨层路由功能。

#### 7. 优点
*   **极度轻量**：仅增加约 0.1% 的额外参数，对推理延迟影响极小。
*   **结构创新**：将微调视角从“层内变换”扩展到了“跨层路由”，为解决灾难性遗忘和推理退化提供了新思路。
*   **即插即用**：作为 PEFT 的补充，可以轻松集成到现有的 VLM 微调框架中。

#### 8. 不足与局限
*   **模型泛化验证不足**：实验主要集中在 Qwen3-VL-2B 上，尚未在更大规模（如 7B, 72B）或其他架构（如基于 Llama 的 VLM）上验证通用性。
*   **模态纠缠风险**：在极深层，视觉和文本表征可能高度融合，此时简单的二元模态掩码可能不再是最优选择。
*   **长文本/多图场景**：实验未深入探讨在极长序列或多图交织任务下，全局池化 Query 是否会丢失局部细节信息。
