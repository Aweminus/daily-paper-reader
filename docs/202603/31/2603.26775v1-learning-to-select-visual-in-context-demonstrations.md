## 摘要
多模态大语言模型 (MLLMs) 通过上下文学习 (ICL) 适应视觉任务，这在很大程度上依赖于示例的质量。主流的示例选择策略是无监督的 k-最近邻 (kNN) 搜索。尽管这种“相似性优先”的方法很简单，但对于复杂的客观事实回归任务来说并非最优；它会选择冗余的示例，无法捕捉任务完整的输出范围。我们将选择过程重新定义为一个序列决策问题，并提出了“学习选择示例” (LSD)，通过训练强化学习智能体来构建最优示例集。利用带有以查询为中心的 Transformer 解码器的 Dueling DQN，我们的智能体学习了一种能够最大化 MLLM 下游性能的策略。通过在五个视觉回归基准测试上进行评估，我们发现了一个关键的二分现象：虽然 kNN 在主观偏好任务中仍然是最优的，但 LSD 在客观事实回归任务上显著优于基准方法。通过平衡视觉相关性与多样性，LSD 能够更好地定义回归边界，阐明了在何种情况下视觉 ICL 严格需要学习型选择。

## Abstract
Multimodal Large Language Models (MLLMs) adapt to visual tasks via in-context learning (ICL), which relies heavily on demonstration quality. The dominant demonstration selection strategy is unsupervised k-Nearest Neighbor (kNN) search. While simple, this similarity-first approach is sub-optimal for complex factual regression tasks; it selects redundant examples that fail to capture the task's full output range. We reframe selection as a sequential decision-making problem and introduce Learning to Select Demonstrations (LSD), training a Reinforcement Learning agent to construct optimal demonstration sets. Using a Dueling DQN with a query-centric Transformer Decoder, our agent learns a policy that maximizes MLLM downstream performance. Evaluating across five visual regression benchmarks, we uncover a crucial dichotomy: while kNN remains optimal for subjective preference tasks, LSD significantly outperforms baselines on objective, factual regression tasks. By balancing visual relevance with diversity, LSD better defines regression boundaries, illuminating when learned selection is strictly necessary for visual ICL.

### 论文总结：《Learning to Select Visual In-Context Demonstrations》

#### 1. 核心问题与整体含义
*   **研究动机**：多模态大语言模型（MLLM）的上下文学习（ICL）性能高度依赖于所选示例（Demonstrations）的质量。
*   **核心问题**：目前主流的示例选择方法是基于特征相似性的无监督 **kNN 搜索**。作者指出，这种“相似性优先”的策略在处理**复杂的客观事实回归任务**（如年龄估计、图像质量评估）时表现不佳。原因在于 kNN 选出的示例往往高度冗余，无法覆盖任务的完整输出范围（例如，预测 8 岁小孩的年龄时，kNN 只选其他小孩，而忽略了成年人作为对比边界的作用）。
*   **研究背景**：研究重心已从训练少样本学习器转向如何通过有效的提示（Prompting）来适配大规模预训练模型。

#### 2. 方法论：LSD (Learning to Select Demonstrations)
*   **核心思想**：将 K-shot 示例选择建模为一个**序列决策问题**（MDP），利用强化学习（RL）训练一个智能体，使其能够根据当前查询（Query）和已选示例，迭代地构建最优示例集。
*   **关键技术细节**：
    *   **状态表示（State）**：包含查询图像的嵌入向量和已选示例的序列。
    *   **以查询为中心的 Transformer 解码器**：为了防止智能体出现“策略坍缩”（即对任何查询都选择相同的一组通用示例），设计了专门的解码器，将查询向量作为 Target，已选示例作为 Memory 进行交叉注意力计算，确保选择策略与查询高度相关。
    *   **Dueling DQN 架构**：将 Q 值分解为状态价值 $V(s)$ 和动作优势 $A(s, a)$。为了处理海量动作空间（整个数据集 $N$ 个样本），优势函数通过查询向量与候选样本嵌入的内积来近似。
    *   **大规模动作空间优化**：引入 **FAISS (近似最近邻搜索)**。智能体输出一个“优势查询向量”，通过 FAISS 在毫秒级时间内从数万个候选样本中检索出最相关的候选子集，从而将 $O(N)$ 的计算复杂度降至对数级。
    *   **奖励函数**：$r_t = R(s_{t+1}) - R(s_t)$，其中 $R$ 是 MLLM 在当前示例集下的负平均绝对误差（-MAE）。这种稀疏奖励鼓励智能体选择能显著提升预测精度的样本。

#### 3. 实验设计
*   **数据集**：涵盖五大视觉回归基准：
    1.  **UTKFace**：年龄预测（客观事实）。
    2.  **KonIQ-10k / KADID-10k**：图像质量评估（客观事实/人工失真）。
    3.  **AVA**：美学评分（主观偏好）。
    4.  **SCUT-FBP5500**：颜值评分（主观偏好）。
*   **Benchmark 与对比方法**：
    *   **Random**：随机选择示例。
    *   **kNN**：基于 SigLIP 特征相似度的最近邻搜索。
    *   **0-Shot**：无示例基准。
    *   **Supervised (Sup.)**：全监督训练的 SOTA 性能作为参考上限。
*   **测试模型**：主要使用 Gemma 3 4B-it 产生奖励信号，并在 Qwen 2.5 7B 和 Phi-3.5-vision 上验证泛化性。

#### 4. 资源与算力
*   **硬件**：单张 **NVIDIA A100 GPU**。
*   **训练时长**：训练 16,000 步大约需要 **7 小时**。
*   **效率优化**：通过预计算 SigLIP 嵌入和使用 FAISS 索引，显著降低了训练和推理时的计算开销。

#### 5. 实验数量与充分性
*   **实验规模**：
    *   在 5 个不同性质的数据集上进行了完整测试。
    *   对比了不同的 K 值（1, 4, 8, 16）。
    *   **跨模型泛化实验**：用 Gemma 训练的策略直接用于 Qwen 和 Phi 模型，验证了策略的通用性。
    *   **跨数据集泛化实验**：用年龄预测训练的策略去跑画质评估任务。
    *   **消融实验**：验证了“以查询为中心”的解码器架构对防止策略坍缩的必要性。
*   **评价**：实验设计非常充分且客观。作者不仅关注定量指标（MAE），还通过定性分析（可视化示例选择过程）解释了 LSD 为什么有效，并诚实地指出了在主观任务上的局限性。

#### 6. 主要结论与发现
*   **任务二分法**：LSD 在**客观事实回归任务**（年龄、画质）上显著优于 kNN，因为它能平衡相关性与多样性，选出定义“回归边界”的示例。而在**主观偏好任务**（美学、颜值）中，kNN 依然是最优的，因为这类任务更依赖视觉上的相似锚点。
*   **隐含标签意识**：尽管 LSD 在选择时无法看到候选样本的标签，但通过优化 MLLM 的反馈，它“进化”出了选择标签空间中接近查询样本的能力。
*   **集合优于顺序**：实验发现，示例的**选择集合**比其排列顺序对性能的影响更大。
*   **通用性**：LSD 学习到的是一种通用的检索启发式策略（如选择多样化的锚点），可以跨模型迁移。

#### 7. 优点
*   **架构创新**：成功将大规模离散动作空间的强化学习应用于视觉 ICL 示例选择，解决了 $O(N)$ 搜索难题。
*   **深入的洞察**：揭示了 ICL 策略与任务属性（客观 vs 主观）之间的深层联系，而非盲目追求一种万能算法。
*   **高效性**：训练时间短，且推理时利用 FAISS 保证了实用性。

#### 8. 不足与局限
*   **任务覆盖面**：目前主要集中在视觉回归任务，尚未在分类、目标检测或更复杂的视觉推理任务上验证。
*   **主观任务表现**：在美学评分等主观任务上，学习型策略未能超越简单的 kNN，说明对于这类任务，视觉相似性可能确实是第一准则。
*   **依赖性**：性能仍受限于底层视觉编码器（如 SigLIP）的表征能力。
